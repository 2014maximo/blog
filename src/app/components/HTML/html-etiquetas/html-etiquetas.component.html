PROPIEDADES DE LAS CAPAS

<div data-backdrop="static"></div>
El atributo de fondo de datos especifica si el modal debe tener una superposición oscura (el color de fondo de la página actual) o no.

POSIBLES VALORES
- 'true' (Muestra el manto gris de fondo y al hacer click cierra modal),
- 'false' (No muestra el manto gris, por lo tanto no se puede cerrar modal),
- 'static' (Muesra el manto gris, pero no cierra el modal al hacer click sobre él).
----------------------------------------------------------

<div data-keyboard="false"></div>
El atributo de fondo de datos especifica si el modal debe tener una superposición oscura (el color de fondo de la página actual) o no.

POSIBLES VALORES
- 'true' (Muestra el manto gris de fondo y al hacer click cierra modal),
- 'false' (No muestra el manto gris, por lo tanto no se puede cerrar modal).
----------------------------------------------------------

<div tabindex="-1"></div>
Significa que el elemento debe ser enfocado, pero no debe de ser accesible a través de la navegación secuencial del teclado. El valor que se le ingresa al atributo es el orden de navegación cada que se presiona la tecla "TAB"

POSIBLES VALORES
- '-1' Se da este valor para dar prioridad en el orden de la navegación.
- '0 < $n' Mientras se va presionando la tecla "TAB" va navegando en forma ascendente según este valor en cada elemento.
----------------------------------------------------------
