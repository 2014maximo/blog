<meta>

Las etiquetas <meta> son herramientas esenciales para optimizar el rendimiento de una página web, mejorar su visibilidad en los motores de búsqueda y proporcionar información crucial a los navegadores y otros programas. Al utilizar correctamente las etiquetas <meta> y sus atributos, puedes asegurarte de que tu sitio web funcione de manera eficiente y cumpla con sus objetivos.

El atributo charset declara la codificación de caracteres.

Los documentos HTML modernos deberían usar siempre UTF-8, aunque no sea un requisito. En HTML, la etiqueta <meta> no requiere una etiqueta de cierre.

Son cruciales para varios aspectos del funcionamiento de una página web:

    1

  • El atributo name es obligatorio para todas las etiquetas <meta> y define el tipo de información meta que se proporciona. Algunos valores comunes para el atributo name incluyen:
    • author: Especifica el autor del documento web.
    • description: Proporciona una breve descripción del contenido de la página.
    • keywords:

      Define palabras clave relevantes para el contenido de la página.


    • charset: Indica la codificación de caracteres del documento web.
    • viewport: Controla cómo se representa la página web en dispositivos móviles.
    • robots: Instruye a los motores de búsqueda sobre cómo indexar y rastrear la página.

    2

  • Atributo content:
    proporciona el valor de la información meta especificada por el atributo name. Por ejemplo, para una etiqueta <meta> con name="description", el atributo content contendría la descripción de la página por ejemplo:

    • 3

  • Atributos adicionales:
    Además de los atributos name y content, las etiquetas <meta> pueden tener varios atributos opcionales que proporcionan información adicional, como:
    • http-equiv: Especifica un metadato HTTP equivalente para navegadores más antiguos.
    • scheme: Define el esquema de identificación para el metadato.
    • lang: Indica el idioma del metadato.

<b> ó <strong>

¿Cuál es la diferencia?
La semántica. '<strong>' se utiliza para indicar que el texto es fundamental o semánticamente importante para el texto que lo rodea, mientras que '<b>' no indica tal importancia y simplemente representa texto que debe ir en negrita.

Si utilizara '<b>', un programa de conversión de texto a voz no pronunciaría la(s) palabra(s) de forma diferente a las demás palabras que la(s) rodean: simplemente está llamando la atención sobre ellas sin añadirles ninguna importancia adicional.
Sin embargo, al utilizar '<strong>', el mismo programa querría pronunciar esas palabras con un tono de voz diferente para transmitir que el texto es importante de alguna manera.

<del> ó <s>

En cuanto a la apariencia visual (ambas muestran texto tachado), pero difieren en su semántica y propósito de uso.

Diferencias:

  • Semántica:
    • '<s>': Indica texto tachado que ya no es relevante o correcto, pero sin implicar una eliminación formal.
    • '<del>': Indica texto que ha sido eliminado del documento, normalmente en el contexto de revisiones o ediciones.
  • Uso común:
    • '<s>': Para precios anteriores, correcciones, o contenido desactualizado.
    • '<del>': Para mostrar texto eliminado en versiones revisadas de documentos o contenido.
  • Atributos:
    • '<s>': Generalmente no se utilizan atributos adicionales.
    • '<del>': Puede incluir el atributo datetime para especificar cuándo fue eliminado el texto.

Estilos por defecto:

<sup> ó <sub>

Se utilizan para formatear texto como superíndice y subíndice, respectivamente. Estas etiquetas son útiles para representar exponentes, notaciones científicas, fórmulas matemáticas, referencias y otros tipos de texto especializado que requieren una posición elevada o inferior en relación con el texto normal.

<sup> (Superscript / Superíndice)

Se utiliza para elevar el texto ligeramente por encima de la línea base del texto normal. Es comúnmente usada para representar exponentes en matemáticas y ciencias, así como para notaciones específicas como marcas registradas o referencias bibliográficas.

<sub> (Subscript / Subíndice)

Se utiliza para bajar el texto ligeramente por debajo de la línea base del texto normal. Es comúnmente usada para notaciones químicas, índices matemáticos y cualquier otro contexto donde se necesiten subíndices.

Estilos por defecto:

href

Se utiliza para redirigir o apuntar a un URL absoluta o relativa. Como ejemplo una URL Absoluta: https://example.com, y una relativa se refiere a otra sección o directorio dentro del mismo sitio web que la expone.

Estilos por defecto:

hreflang

El atributo hreflang en HTML se utiliza para especificar el idioma y la región de un documento vinculado. Es especialmente útil en el contexto de SEO (Optimización para Motores de Búsqueda) y de sitios web multilingües, ya que ayuda a los motores de búsqueda a entender qué versión de una página mostrar a los usuarios en función de su idioma y región preferidos.

Se utiliza dentro de etiquetas '<link>' en la sección '<head>' de un documento HTML para indicar la relación entre diferentes versiones lingüísticas de una página. A continuación se muestra la estructura básica:

rel

El atributo rel en etiquetas HTML se utiliza para definir la relación entre el documento actual y el recurso vinculado. Es comúnmente utilizado en la etiqueta '<link>' para enlaces a recursos externos, como hojas de estilo, iconos, o archivos de diferentes versiones del mismo documento, y también en la etiqueta '<a>' para describir la relación entre el documento actual y la URL destino.

target

Se utiliza para especificar dónde se abrirá el documento vinculado cuando el usuario haga clic en un enlace o envíe un formulario. Este atributo es comúnmente usado en las etiquetas '<a>' (para enlaces) y '<form>' (para formularios).

Es útil para controlar cómo se abren los enlaces y formularios, proporcionando flexibilidad en la navegación y en la presentación de contenido. Su uso adecuado puede mejorar la experiencia del usuario y contribuir a la seguridad del sitio web.

NOTA:
Cuando se utiliza target="_blank", es recomendable añadir rel="noopener noreferrer" al enlace para mejorar la seguridad. Esto evita que la nueva página tenga acceso a la ventana original a través de window.opener, lo que puede prevenir ataques de tipo "tabnabbing".

title

Se utiliza para proporcionar información adicional sobre un elemento. Este atributo se puede añadir a casi cualquier etiqueta HTML y suele ser empleado para mostrar un texto emergente (tooltip) cuando el usuario pasa el cursor sobre el elemento. El texto del atributo title aparece como un cuadro emergente en la mayoría de los navegadores.

download

Se utiliza en la etiqueta '<a>' para indicar que el navegador debe descargar el archivo enlazado en lugar de navegar hacia él. Este atributo permite especificar un nombre de archivo sugerido para guardar el archivo descargado.
Proporciona un enlace que permite a los usuarios descargar un archivo directamente en lugar de abrirlo en una nueva pestaña o ventana del navegador.
