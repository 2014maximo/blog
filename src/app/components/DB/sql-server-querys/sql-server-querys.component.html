PROCEDIMIENTOS DE ALMACENADO

Son procesos que llevan a cabo varios pasos DDL ó DML y se ejecutan todos al llamar al título del procedimiento.

1. En este caso, este procedimiento requiere un "id" como parámetro y lo que realiza simplemente es traer dos tipos de campos según el "id" solicitado.

CREATE PROCEDURE PDA_ListaCamas
@idCama int

AS BEGIN
SELECT nombre, descripcion FROM listaCamas
WHERE idCama = @idCama
END

2. En este caso, este procedimiento requiere un "id" como parámetro y lo que realiza simplemente es traer dos tipos de campos según el "id" solicitado.

CREATE PROCEDURE PDA_guardarActualizarTipoHabitacion
@id int,
@nombre varchar(50),
@descripcion varchar(150)

AS BEGIN
if @id = 0
INSERT INTO listarTipoHabitacion(nombre, descripcion)
VALUES (@nombre, @descripcion)
else
UPDATE listarTipoHabitacion
SET nombre = @nombre, descripcion = @descripcion
WHERE id = @id
END

3. En este caso, este procedimiento requiere un "id" como parámetro y lo que realiza simplemente es traer dos tipos de campos según el "id" solicitado.

CREATE PROCEDURE PDA_eliminarTipoHabitacion
@id int

AS BEGIN
DELETE FROM listarTipoHabitacion
WHERE id = @id
END
