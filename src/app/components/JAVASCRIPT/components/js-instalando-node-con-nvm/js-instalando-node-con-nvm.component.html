Node JS como todo en el mundo de desarrollo tiene cambios de alto impacto, no por eso hay que parar y bararse en versiones viejas o a veces por las nuevas, nuestros desarrollos anteriores se entorpecen.
Aquí es cuando aparece una solución muy recursiva que funciona como un suiche entre versiones, denominado "NVM"

NVM PARA WINDOWS

NVM solo esta disponible en Linux y macOS, pero existe una versión separada de la versión oficial llamada "nvm-windows" con limitaciones pero que permite usar las utilidades principales y más prácticas.

COMANDOS

nvm list

Mostrar las versiones instaladas utilizables.

nvm install v12.20.2

Instalar la versión descrita.

nvm use 12.20.2

Usar la versión descrita.
