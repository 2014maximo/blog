VARIABLES Y TIPOS DE DATOS

MÉTODOS

(onBlur)

Se ejecuta cuando se cambia el foco de trabajo actual a otro punto, como por ejemplo pasar de un input de nombre al input de apellido.

(onLoad)

Ejecuta el script después de que las páginas terminen de cargarse.

(onFocus)

Ejecuta el script cuando el elemento recibe el foco (por ejemplo, cuando un cuadro de texto está activo).

(onChange)

Ejecuta el script cuando se cambia el valor de un elemento.

MAP

Recolectar todos los atributos de un array de objetos.
