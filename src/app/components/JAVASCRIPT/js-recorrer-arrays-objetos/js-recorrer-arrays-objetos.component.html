SPLICE

Este método elimina elementos de conteo de la matriz, comenzando en el índice especificado. Los elementos eliminados se devuelven como resultado del método.
Algo muy importante de esta función y que parece nunca te explican, es que si inmediatamente usas el "SPLICE" y le pasas el valor a una variable, solo le va a pasar el elemento eliminado. Solo después de terminar el proceso "SPLICE", el array al que le aplicaste el "SPLICE" cambió y ahora esta sin el elemento eliminado.

concat('otherArray')

Este método devuelve un nuevo array que concatena el array sobre el que se ha llamado con el array especificado como argumento. Se pueden especificar varias matrices.

join('separator')

Este método une todos los elementos de la matriz para formar una cadena. El argumento especifica el carácter utilizado para delimitar los elementos.

pop()

Este método elimina y devuelve el último elemento de la matriz

shift()

Este método elimina y devuelve el primer elemento de la matriz.

push()

Este método agrega el elemento especificado al final de la matriz.

unshift(item)

Este método inserta un nuevo elemento al comienzo de la matriz.

reverse()

Este método devuelve una nueva matriz que contiene los elementos en orden inverso.

slice(start, end)

Este método devuelve una sección de la matriz.

sort()

Este método ordena la matriz. Se puede utilizar una función de comparación opcional para realizar comparaciones personalizadas.

unshift(item)

Este método inserta un nuevo elemento al comienzo de la matriz.

every(test)

Este método llama a la función de prueba para cada elemento de la matriz y devuelve verdadero si la función devuelve verdadero para todos ellos y falso de lo contrario.

some(test)

Este método devuelve verdadero si llamar a la función de prueba para cada elemento de la matriz devuelve verdadero al menos una vez.

filter(test)

Este método devuelve una nueva matriz que contiene los elementos para los que la función de prueba devuelve verdadero.

find(test)

Este método devuelve el primer elemento de la matriz para el que la función de prueba es verdadera.

findIndex(test)

Este método devuelve el índice del primer elemento de la matriz para el que la función de prueba devuelve verdadero.

foreach(callback)

Este método invoca la función callback para cada elemento de la matriz, como se describe en la sección anterior.

includes(value)

Este método devuelve true si el array contiene el valor especificado.

map(callback)

Este método devuelve un nuevo array que contiene el resultado de invocar la función callback para cada elemento del array.

reduce(callback)

Este método devuelve el valor acumulado producido al invocar la función callback para cada elemento del array.
