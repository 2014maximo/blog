Última actualización:

4 pasos para la instalación de un proyecto de ANGULAR, adicional estan sus versiones y características, editores recomendados, características de las versiones de NODE-JS como tambien los comandos más utilizados de ANGULAR CLI.

1

Instalar el angular cli que es una herramienta de gestión del framework de angular, se instala por consola y es recomendado estar como administrador. Copiar el siguiente comando:

npm install -g @angular/cli

COMANDOS ANGULAR CLI

1. Crear un proyecto Angular nuevo para empezar a desarrollar.

ng new nombre_aplicacion

2. Compila el proyecto y lo expone en una url para visualizarlo en un navegador.

ng serve

3. Lo mismo que el comando anterior pero abre el proyecto automáticamente en el navegador.

ng serve o

4. Compila el proyecto en un puerto específico, en este caso saldría por el "localhost:4201".

ng serve --port 4201

5. Crea un conjunto de 3 tipos de archivos: nombredelcomponente.html, nombredelcomponente.ts, nombredelcomponente.spec.ts.

ng component ruta/nombredelcomponente

6. Lo mismo que lo anterior sin el archivo de pruebas.spec.ts.

ng g c nombre/componente --skip-tests

7. Lo mismo que lo anterior sin el archivo de estilos

ng g c nombre/componente --skip-tests --inline-style

8. Crea un archivo.service.ts con la extructura de un archivo de servicios de Angular base para desarrollar.

ng g s ruta/nombreArchivoService

9. Actualiza la version de "Angular CLI".

ng update @angular/cli @angular/core

3

Instalar Node JS
se recomienda la versión LTS. Node JS es un entorno en tiempo de ejecución multiplataforma, basado en Javascript, en realidad esta descripción es compleja de entender a primera vista, tal vez basta con describir que es un programa para cualquiera de los sistemas operativos, que permite instalar a través de comandos un proyecto base dentro de una carpeta con todos los archivos necesarios para un desarrollo, en este caso Angular. Entre otras muchas cosas o frameworks.

4

Una vez reunidas las anteriores herramientas esta todo listo para tirar el comando:

ng new nombre_aplicacion

Con este comando se instala angular en la carpeta que se requiera generar nuestro nuevo proyecto.
Ya de preferirse se puede optar por alguna versión específica según ser requiera con sus características a continuación se muestra, por defecto el Angular CLI instala siempre la última.
