Instalación

1

Instalar el angular cli, es una herramienta de gestión del framework de angular, se instala por consola y es recomendado estar como administrador. Copiar el siguiente comando:

La última - primera vez

npm install -g @angular/cli

Instalar una versión específica

npm install -g @angular/cli@8

Pasos para hacer un downgrade

npm uninstall -g @angular/cli

npm cache clean

npm install -g @angular/cli@1

actualizar

ng update @angular/cli @angular/core

actualizar a la mayor versión

ng update @angular/cli@^ @angular/core@^

actualizar a versiones específicas

ng update @angular/cli@8 @angular/core@8

CREAR UN NUEVO PROYECTO

Para finalizar una vez reunidas las anteriores herramientas, esta todo listo para tirar el comando:

Con este comando se instala angular en la carpeta que se requiera generar nuestro nuevo proyecto. Ya de preferirse se puede optar por alguna versión específica según ser requiera con sus características a continuación se muestra, por defecto el Angular CLI instala siempre la última.
