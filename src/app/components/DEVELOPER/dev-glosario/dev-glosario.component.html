ANALYTICS
(Google analytics)

Herramienta de google utilizada para medir datos de los desarrollos, principalmente con fines comerciales para medir tendencias y el comportamiento del público frente a lo que más se mueve del momento, lo que esta a la moda y es el foco del subconciente general, para sacarle provecho comercial.

CLOUD FIRESTORE
Guarda y sincroniza en la nube

Firebase Cloud Firestore es una base de datos documental NoSQL. Almacena, sincroniza y consulta datos de apps móviles y web a escala global.

FIREBASE CONSOLE

Es una suite de herramientas integrada con servicios de Google. Proporciona una plataforma de administración para Android, iOS y Web, pudiendo gestionar a la vez muchas apps de un mismo proyecto.

SDK
(Kit de Desarrollo de Software)

Paquete de herramientas adecuadas para los desarrollos de aplicaciones, que generalmente no es omitible y suele ser muy indispensable la aplicación cuando el lenguaje lo requiere. Uno de los SDK más conocidos es el de Java, también en frameworks de Javascript se utilizan como el SDK de control de "FIREBASE".
