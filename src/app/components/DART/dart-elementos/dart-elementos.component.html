La sintaxis de DART es similar a la mayoría de lenguajes de programación, no esta de más explicar como se aplica en DART, veamos la descripción de los elementos básicos con ejemplos.

DECLARACIÓN DE VARIABLES

Opcional el uso de tipos.

El operador new se utiliza para inicializar una nueva instancia de una clase. Después de inicializar una clase, se llama inmediatamente a su constructor. Los parámetros del constructor se pasan en el momento de la inicialización.

TIPOS FUNDAMENTALES

Tipo Descripción Ejemplo
int Números enteros 7, -19, 0
double Números en coma flotante 6.2543, -5.7, 0.00
String Cadenas "Hola mundo", "h"
bool Booleanos true, false
List Listas [1,2,4], ["hi", "hola"]
Map Mapas { "b": 7, "d": 5 }

OPERADORES ARITMÉTICOS

Operador Descripción Ejemplo
++ Incrementar un entero en 1 b++;
-- Decrementar un entero en 1 b--;
+= Incrementar un entero en una cantidad arbitraria b += 77; // incrementa b por 77
* Multiplicación b = a * c;
/ División b = a / c;
~/ División de enteros b = a ~/ c; // b es 1
+ Adición b = a + c;
- Subtracción b = a - c;
% El restante b = 6 % 4; // b es 2
